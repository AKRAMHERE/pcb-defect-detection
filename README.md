# PCB Defect Detection and Automated Inspection System

**YOLOv8n-based manufacturing quality control pipeline for CPU-only deployment.**

Detects and localizes PCB manufacturing defects in real-time from images, video recordings, and live webcam streams. Generates structured CSV/JSON inspection reports suitable for MES integration.

---

## Defect Classes

| ID | Class | Description |
|----|-------|-------------|
| 0  | `missing_hole` | Drilled hole absent from board |
| 1  | `mouse_bite` | Irregular board edge notching |
| 2  | `open_circuit` | Broken trace/conductor |
| 3  | `short` | Unintended conductor bridge |
| 4  | `spur` | Extra copper protrusion on trace |
| 5  | `spurious_copper` | Unwanted copper deposit |

## Dataset Download

### Option A — Roboflow API (recommended)

1. Create a free account at [roboflow.com](https://app.roboflow.com)
2. Get your API key from **Settings → API**
3. Run:

```bash
pip install roboflow
python scripts/download_dataset.py --api-key YOUR_API_KEY --output data/pcb_dataset
```

**Dataset used:** [PCB Defect Detection — Roboflow Universe](https://universe.roboflow.com/roboflow-100/pcb-defect-detection)

Expected format after download:
```
data/pcb_dataset/
├── train/
│   ├── images/     # ~693 images
│   └── labels/     # YOLO .txt annotations
├── valid/
│   ├── images/     # ~99 images
│   └── labels/
├── test/
│   ├── images/     # ~50 images
│   └── labels/
└── dataset.yaml    # Auto-generated by trainer
```

### Option B — Manual download

1. Go to [universe.roboflow.com](https://universe.roboflow.com) → search "PCB Defect Detection"
2. Download in **YOLOv8** format
3. Place in `data/pcb_dataset/`

### Option C — Scaffold empty structure

```bash
python scripts/download_dataset.py --scaffold-only --output data/pcb_dataset
# Then manually copy your dataset files
```

## Training

```bash
# Default: 50 epochs, batch=8, imgsz=640, CPU
python main.py train --dataset data/pcb_dataset

# Custom config
python main.py train \
  --dataset data/pcb_dataset \
  --epochs 100 \
  --batch 4 \
  --imgsz 640 \
  --run-name pcb_v2

# Best weights auto-saved to: models/pcb_v2_best.pt
```

**CPU Training Time Estimates (Intel i7-class):**

| Epochs | Batch | Approx. Time |
|--------|-------|--------------|
| 50     | 8     | ~3–5 hours   |
| 100    | 4     | ~6–9 hours   |

Tips for faster CPU training:
- Reduce `--imgsz` to `416` (~2× speedup, minor accuracy drop)
- Reduce `--batch` to `4` if memory constrained
- Use `--epochs 30` for a quick sanity-check run

## Inference

### Single image

```bash
python main.py image \
  --input data/pcb_dataset/test/images/pcb_001.jpg \
  --model models/pcb_defect_v1_best.pt \
  --conf 0.25

# With display window
python main.py image --input test.jpg --model models/best.pt --display
```

### Batch directory

```bash
python main.py batch \
  --input data/pcb_dataset/test/images/ \
  --model models/pcb_defect_v1_best.pt \
  --output outputs/
```

### Video file

```bash
python main.py video \
  --input recordings/pcb_line.mp4 \
  --model models/pcb_defect_v1_best.pt \
  --skip-frames 2        # Run inference every 2nd frame (CPU perf)
  --display
```

### Live webcam

```bash
python main.py webcam \
  --model models/pcb_defect_v1_best.pt \
  --camera 0             # Device index (0 = default webcam)
  --conf 0.30

# Keyboard controls:
#   q         → stop
#   s         → save snapshot
#   +/-       → adjust confidence threshold ±0.05
```

### Evaluation

```bash
python main.py eval \
  --dataset data/pcb_dataset \
  --model models/pcb_defect_v1_best.pt
```

### Demo (no dataset needed)

```bash
python main.py demo --model models/best.pt
# Runs on synthetic PCB-like image to validate installation
```

## Python API

```python
from src import PCBDefectDetector, ImageProcessor, ReportGenerator

# Initialize
detector  = PCBDefectDetector(model_path="models/best.pt", confidence_threshold=0.3)
processor = ImageProcessor(detector=detector, output_dir="outputs/images")
reporter  = ReportGenerator(output_dir="outputs/reports", fail_threshold=0)

# Inspect single image
detections, saved_path = processor.inspect_image("pcb_sample.jpg")

# Batch directory
from src.utils import InspectionSummary
detections, summary = processor.inspect_directory("data/test_images/")

# Generate reports
reporter.generate_full_report(detections, summary)
```
